Let us see why \(\log(U)^* = - \log(U)\) for \(U \in \mathrm{U}(N) \cap b(I, 1)\), where \(\log(U) = \sum_{n=1}^\infty (-1)^{n+1} \frac{(U-I)^n}{n}.\)

Let \(U = VDV^*\) be an eigendecomposition for \(U\), where \(D = \text{diag}(e^{i\theta_1}, \cdots, e^{i\theta_N})\), for \(\theta_1, \cdots, \theta_N \in (-\pi, \pi)\), and \(V \in \mathrm{U}(N)\).

Here is why \(e^{i \theta_j} \ne -1\) for all \(j = 1, \cdots, N\). First note that \((VDV^* - I) = V(DV^* - V^*) = V(D-I)V^*\) by unitarity of \(V\). Hence, \(\lVert I - U \rVert = \lVert I - D \rVert\). We have \(\lVert I - D \rVert^2 = \sum_{jk} |(I-D)_{jk}|^2\). Suppose \(e^{i\theta_m} = -1\) at entry \(mm\) in \(D\). Then \(|(I-D)_{mm}|^2 = |2|^2 = 4.\) Then we have \[\begin{align*} \lVert I - D \rVert^2 = \sum_{jk}|(I-D)_{jk}|^2 \ge 4. \end{align*}\] But \(U \in b(I,1)\), and we have a contradiction. This will be helpful when we consider the principal branch of the logarithm, which is defined on \(\C \setminus (-\infty, 0]\).

For seeing how matrices \(V\) pass through the logarithm, we note that \((V(D-I)V^*)^n = V(D-I)^nV^*\), for \(n\) a positive integer. Then we have \[\begin{align*} \log(VDV^*) &= \lim_{N \to \infty} \sum_{n=1}^N (-1)^{n+1} \frac{V(D-I)^nV^*}{n}\\ &= V \log(D) V^*. \end{align*}\]

We know that \(\log(D) = \text{diag} (\log (\lambda_1), \cdots, \log(\lambda_N))\), where \(\lambda_j = e^{i \theta_j}\). Using the principal branch of the logarithm, and employing the fact that \(\theta_j \ne \pi\) (since \(U \in b(I, 1)\)), we have \(\log(\lambda_j) = i \theta_j\) for all \(j=1, \cdots, N\).

Note that \(\log(D)^* = \text{diag}(-i\theta_1, \cdots, -i\theta_n)\), which, in turn, is equal to \(-\log(D)\).

Therefore, we have \[\begin{align*} \log(U)^* &= (V \log(D) V^*)^* \\ &= V \log(D)^* V^* \\ &= -V \log(D) V^* \\ &= - \log(U). \end{align*}\]

Therefore, \(\log(U)\) is skew-Hermitian whenever \(U \in \mathrm{U}(N) \cap b(I, 1)\).

We want to show that \(\lVert H_i \rVert \le \eta\) for all \(i \in [3]\) implies that \(|\Tr(H_1H_2H_3)| < \eta^3.\)

Lemma. The function \(\langle A, B \rangle = \Tr (A^*B)\) defines an inner product on the \(\R\) or \(\C\) vector space \(\C^{N \times N}\).

Proof. Let us consider first \(\C^{N \times N}\) as a \(\C\) vector space. We leave the \(\R\) proof for another life. Let us check the axioms in definition 2.1 on page 9 of Alt (2012), except that we will use anti-linearity in the first slot (the physicist’s convention).

Let \(\alpha \in \C\) and consider \(A, B \in \C^{N \times N}\). We have \[\langle \alpha A, B \rangle = \Tr((\alpha A)^* B) = \alpha^* \Tr(A^* B) = \alpha^* \langle A, B \rangle.\]

We also have \[\langle A, \alpha B \rangle = \Tr((A^* (\alpha B)) = \alpha \langle A, B \rangle.\]

And if \(A,B,C \in \C^{N \times N}\), we have \[\langle A + C, B \rangle = \Tr((A+C)^* B) = \Tr(A^*B + C^*B) = \Tr(A^*B) + \Tr(C^*B) = \langle A, B \rangle + \langle C, B \rangle.\]

Similarly, \[\langle A, C+B \rangle = \langle A, B \rangle + \langle A, C \rangle.\] Up to this point we have shown that \(\langle \, \cdot \, , \, \cdot \, \rangle\) is a sesequilinear form.

Next, for \(A, B \in \C^{N \times N}\), we have \[\begin{equation*} \begin{aligned} \overline{\langle A, B \rangle} = \overline{\Tr(A^*B)} = \sum_{i=1}^N \overline{(A^*B)_{ii}} \\ = \sum_{i=1}^N \sum_{j=1}^N \overline{(A^*)_{ij} B_{ji}} = \sum_{i=1}^N \sum_{j=1}^N \overline{\overline{(A_{ji})} B_{ji}} \\ = \sum_{i=1}^N \sum_{j=1}^N (A_{ji}) \overline{B_{ji}} = \sum_{i=1}^N \sum_{j=1}^N (B^*)_{ij} (A_{ji}) \\ = \sum_{i=1}^N (B^* A)_{ii} = \Tr(B^*A) = \langle B, A \rangle, \end{aligned} \end{equation*}\] which shows symmetry.

Next, for \(A \ne 0 \in \C^{N \times N}\), we have \[\begin{align*} \langle A, A \rangle = \Tr(A^* A) = \sum_{i,j=1}^N \lVert A_{ij} \rVert^2 > 0. \end{align*}\] Since \(\langle 0, 0 \rangle = 0\), this shows that \(\langle \, \cdot \, , \, \cdot \, \rangle\) is positive definite. Therefore \(\langle \, \cdot \, , \, \cdot \, \rangle\) defines an inner product on \(\C^{N \times N}\). QED.

Lemma. Let \(X_1, \cdots, X_k\) be \(k \ge 2\) matrices in \(\C^{N \times N}\). Then \[|\Re \Tr X_1 \cdots X_k| \le \lVert X_1 \rVert \cdots \lVert X_k \rVert,\] where \(\lVert \, \cdot \, \rVert\) denotes the Hilbert Schmidt norm induced by the inner product \(\langle A ,B \rangle = \Tr(A^*B)\) defined in the above lemma. That is, \(\lVert \, \cdot \, \rVert = \sqrt{\langle \, \cdot \, , \, \cdot \, \rangle}.\)

Remark. This result does not hold when \(k=1\). The idea is that the square root of \(N\) term in the inequality \(|\cdot|_1 \le \sqrt{N} |\cdot |_2\) is actually needed. For an example, consider the matrix \(X_1 = \begin{bmatrix}0.5 & 0 \\ 0 & 0.5 \end{bmatrix} \in \C^{2 \times 2}\). Then \(\Re \Tr X_1 = 1\). On the other hand, \(\lVert X_1 \rVert = \sqrt{0.5}\).

Proof. We proceed by induction. The base case is \(k=2\). Suppose \(z = \Tr X_1X_2\). Then \(|\Re z| \le |z|\). Therefore \(|\Re \Tr X_1X_2 | \le |\Tr X_1 X_2|\). Let \(X = X_1^*\), so that \(X^* = X_1\). Then we have \(|\Tr X_1 X_2| = |\langle X, X_2 \rangle|\). By the Cauchy Schwarz inequality, we have \(|\langle X, X_2 \rangle| \le \lVert X \rVert \lVert X_2 \rVert\). But \(\lVert X \rVert = \lVert X_1 \rVert\) because \(\lVert X \rVert^2 = \langle X, X \rangle = \Tr X^* X = \Tr X X^* = \Tr X_1^* X_1 = \langle X_1, X_1 \rangle = \lVert X_1 \rVert^2\). Tracing back the steps, we have \(| \Re \Tr X_1 X_2 | \le \lVert X_1 \rVert \lVert X_2 \rVert.\) This settles the base case.

For the inductive step, assume that we have proven the claim for some \(k \ge 2\). Let \(X_1, \cdots, X_{k+1}\) be given. Observe that \(X_1 \cdots X_k \in \C^{N \times N}\) so we can treat it as one matrix. That is, let \(X = X_1 \cdots X_k\). Then we have \(|\Re \Tr X_1 \cdots X_{k+1} | = | \Re \Tr X X_{k+1} |\). The latter term satisfies \(| \Re \Tr X X_{k+1} | \le \lVert X \rVert \lVert X_{k+1} \rVert\) by the base case. Then by an induction argument applied to the inequality \(\lVert Z Y \rVert \le \lVert Z \rVert \lVert Y \rVert\) for matrices \(X,Y\), we obtain \(\lVert X \rVert \le \lVert X_1 \rVert \cdots \lVert X_k \rVert\). Therefore, \[ |\Re \Tr X_1 \cdots X_{k+1} | \le \lVert X_1 \rVert \cdots \lVert X_k \rVert \lVert X_{k+1} \rVert, \] which completes the induction step. Thus, the lemma is proved. QED.

Assumptions for the following lemmas. Let \(p \in \mathcal{P}(\Lambda_n)\) be a plaquette. Let \(U \in \mathrm{U}(N)^{E_n}\) be a unitary configuration. Let \(U_1 = U_{p_1}\), \(U_2 = U_{p_2}\), \(U_3= U_{p_3}\), and \(U_4 = U_{p_4}\). Suppose \(H_1, H_2, H_3, H_4 \in \mathrm{H}(N)\) satisfy \(e^{iH_1} = U_{p_1}\), \(e^{iH_2} = U_{p_2}\), \(e^{iH_3} = U_{p_3}\), and \(e^{iH_4} = U_{p_4}\). Suppose that \[ H_p = H_1 + H_2 - H_3 - H_4. \] Let \[\begin{equation} \begin{aligned} R_1 &= I + iH_1 - H_1^2/2 \\ R_2 &= I + iH_2 - H_2^2/2 \\ R_3 &= I - iH_3 - H_3^2/2 \\ R_4 &= I - iH_4 - H_4^2/2 \\ \end{aligned} \end{equation}\] and suppose \[ R_p = R_1R_2R_3R_4. \] Suppose that \(\lVert H_i \rVert \le \eta \le 1\) for all \(i = 1, 2, 3, 4\). Suppose also that \(\phi(X) = \Re \Tr (I - X)\) for any \(X \in \C^{N \times N}\).

Lemma. We have \[ \left|\phi(R_p) - \frac{1}{2} \lVert H_p \rVert^2 \right| \le C \eta^3, \] where the constant \(C > 0\) depends only on \(N\).

Proof. First we expand \(\phi(R_p)\), which satisfies \[\begin{equation*} \begin{aligned} \phi(R_p) = \Re \Tr (I - (I + iH_1 - H_1^2/2) \times \\ (I + iH_2 - H_2^2/2) \times (I - iH_3 - H_3^2/2) \\ \times (I - iH_4 - H_4^2/2) ). \end{aligned} \end{equation*}\] The product \(R_p = R_1 R_2 R_3 R_4\) will have \(3 \times 3 \times 3 \times 3 = 81\) terms, which we classify based on how many matrices \(H_j\) appear in each one.

First we begin with terms without any matrices \(H_j\). There is one term that is equal to \(I\).

Then we move to linear terms with one matrix \(H_j\). There are four terms of the form \(i H_j\) up to a sign. The contribution is \(X_{\text{lin}} := i(H_1 + H_2 - H_3 - H_4).\)

Then we move to the quadratic terms with two matrices \(H_j\). There are \(\binom{4}{2} = 6\) mixed terms of the form \(zH_kH_j\) where \(k < j\) and \(z \in \{-1,1\}\), and then \(4\) homogeneous terms of the form \(-\frac{1}{2}H_j^2\). The contribution is \[ X_{\text{quad}} := -\frac{1}{2}(H_1^2 + H_2^2 + H_3^2 + H_4^2) -H_1H_2 + H_1H_3 + H_1H_4 + H_2H_3 + H_2H_4 - H_3H_4. \]

There are also cubic, quartic, quintic, sextic, septic, and octic terms, but their contribution will be absorbed into the upper bound \(C\eta^3\) after an application of the previous lemma. But let us denote by \(X_{\text{cubic}+} \in \C^{N \times N}\) the sum of these 66 higher order terms.

We have \[ R_p = I + X_{\text{lin}} + X_{\text{quad}} + X_{\text{cubic}+} \] and therefore \[ \Re \Tr ( I - R_p ) = - \Re \Tr (X_{\text{lin}}) - \Re \Tr (X_{\text{quad}}) - \Re \Tr (X_{\text{cubic}+}) \] First we show that \(\Re \Tr (X_{\text{lin}}) = 0\). Recall that the trace of a matrix is equal to the sum of its eigenvalues. Since the eigenvalues of \(H_p = H_1 + H_2 - H_3 - H_4\) are real as \(H_p\) is a Hermitian matrix, it follows that the eigenvalues of \(i(H_1 + H_2 - H_3 - H_4)\) are all imaginary. Taking the real part of the trace, we get zero, as desired.

Next, we show that \[-\Re \Tr (X_{\text{quad}}) = \frac{1}{2}\lVert H_1 + H_2 - H_3 - H_4 \rVert^2.\] Since \(H_p\) is Hermitian, we get that \(H_p = H_p^*\). Therefore \(\lVert H_p \rVert^2\) is equal to \(\Tr(H_p^*H_p) = \Tr H_p^2.\) Note that \(H_p^2\) in expanded form has 16 terms. Note that 12 of these terms are mixed and of the form \(zH_iH_j\) for \(z \in \{-1,1\}\), and 4 are homogeneous of the form \(H_i^2\). In particular, we have \[\begin{equation*} \begin{aligned} H_p^2 = (H_1H_2 - H_1H_3 - H_1H_4 - H_2H_3 - H_2H_4 + H_3H_4) \\ + (H_2H_1 - H_3H_1 - H_4H_1 - H_3H_2 - H_4H_2 + H_3H_4) + \\ + (H_1^2 + H_2^2 + H_3^2 + H_4^2), \end{aligned} \end{equation*}\] where in the second line we’ve written the oppositely ordered terms compared to the first line, and the third line contains the homogeneous terms. We obtain the same value by taking the trace of the first and second lines above on the right hand side because of the identity \(\Tr(AB) = \Tr(BA)\). Therefore, \[\begin{equation*} \begin{aligned} \frac{1}{2} \Tr(H_p^2) = \frac{1}{2} \cdot 2 \Tr(H_1H_2 - H_1H_3 - H_1H_4 - H_2H_3 - H_2H_4 + H_3H_4) \\ + \frac{1}{2} \Tr (H_1^2 + H_2^2 + H_3^2 + H_4^2). \end{aligned} \end{equation*}\] The expression on the left hand side is equal to \(\frac{1}{2} \lVert H_p \rVert^2\), which is a real number. Therefore, applying \(\Re\) to the right hand side of the equation does nothing. That is, \[\begin{align*} \frac{1}{2} \lVert H_p \rVert^2 = \Re \Tr(H_1H_2 - H_1H_3 - H_1H_4 - H_2H_3 - H_2H_4 + H_3H_4)\\ + \frac{1}{2} \Re \Tr (H_1^2 + H_2^2 + H_3^2 + H_4^2). \end{align*}\] Then we recognize that the right hand side is equal to \(-\Re \Tr X_{\text{quad}},\) which is what we wanted to show.

Then to conclude the lemma, we write \[\begin{align*} \left|\phi(R_p) - \frac{1}{2} \lVert H_p \rVert^2 \right| &= \left| \Re \Tr ( I - R_p ) - \frac{1}{2} \lVert H_p \rVert^2 \right| \\ &= \left| - \Re \Tr X_{\text{cubic}+} \right| \\ & \le \sum |\Re \Tr \, zH^{k \ge 3}|. \end{align*}\] where in the last line, the sum is over the 66 higher order terms, and \(H^{k \ge 3}\) stands for a higher order term of the form \(H_{i_1} H_{i_2} \cdots H_{i_k}\) for \(i_1, \cdots, i_k \in \{1,2,3,4\}\) and \(z\) stands for a coefficient which satisfies \(|z| \le 1\). Then by an application of the lemma above and the fact that \(\lVert H_j \rVert \le \eta \le 1\) for all \(j = 1,2,3,4\) and that \(\eta^k \le \eta^3\) whenever \(k \ge 3\), we have \[ \sum |\Re \Tr \, zH^{k \ge 3}| \le 66 \times \eta^3 = C \eta^{3}. \] Note that the constant \(C\) does not really depend on \(N\), but it is easier to say that it depends only on \(N\) rather than to say what it does not depend on. This concludes the proof. QED.

Lemma. We have \[ \left| \phi(U_p) - \frac{1}{2} \lVert H_p \rVert^2 \right| \le C \eta^3, \] where the constant \(C > 0\) depends only on \(N\).

Proof. We want to show that \(\phi(U_p) = \Re \Tr (I - U_p)\) is \(C \eta^3\)-close to \(\frac{1}{2} \lVert H_p \rVert^2.\) But the previous lemma says that \(\frac{1}{2} \lVert H_p \rVert^2\) is \(C\eta^3\) close to \(\phi(R_p)\), so it suffices to prove that \(\phi(R_p)\) is \(C\eta^3\) close to \(\phi(U_p)\), that is, \(|\phi(U_p) - \phi(R_p)| \le C\eta^3\) where \(C\) depends only on \(N\).

We have \[ \begin{aligned} \phi\left(U_p\right)-\phi\left(R_p\right) & =\operatorname{Tr}(I)-\operatorname{Re}\left(\operatorname{Tr}\left(U_p\right)\right)-\operatorname{Tr}(I)+\operatorname{Re}\left(\operatorname{Tr}\left(R_p\right)\right) \\ & =\operatorname{Re}(\operatorname{Tr}(\operatorname{R_p}))-\operatorname{Re} (\operatorname{Tr} (U_p)) . \end{aligned} \]

Since \(\operatorname{Re}\left(z+z_1\right)=\operatorname{Re} z+\operatorname{Re} z_1\) for complex number \(z, z_1\), and \(\operatorname{Tr}(A-B)=\operatorname{Tr}(A)-\operatorname{Tr}(B)\) for \(A, B \in \mathbb{C}^{N\times N}\), we get \[ \phi\left(u_p\right)-\phi\left(R_p\right)=\operatorname{Re}\left(\operatorname{Tr}\left(R_p-U_p\right)\right) . \]

By telescoping, we have \[ \begin{aligned} R_p-U_p & =\left(R_1-U_1\right) R_2 R_3 R_4 \\ & +U_1\left(R_2-U_2\right) R_3 R_4 \\ & +U_1 U_2\left(R_3-U_3^*\right) R_4 \\ & +U_1 U_2 U_3^*\left(R_4-U_4^*\right). \end{aligned} \]

Then by additivity of \(\operatorname{Re}(\cdot)\) and \(\operatorname{Tr}(\cdot)\), we have \[ \begin{aligned} \operatorname{Re} \operatorname{Tr}( R_p-U_p )& = \operatorname{Re} \operatorname{Tr}(\left(R_1-U_1\right) R_2 R_3 R_4) \\ & +\operatorname{Re} \operatorname{Tr}(U_1\left(R_2-U_2\right) R_3 R_4) \\ & +\operatorname{Re} \operatorname{Tr}(U_1 U_2\left(R_3-U_3^*\right) R_4) \\ & +\operatorname{Re} \operatorname{Tr}(U_1 U_2 U_3^*\left(R_4-U_4^*\right)). \end{aligned} \]

Then we obtain by an application of Lemma ? that \[ \begin{aligned} \left|\phi\left(U_p\right)-\phi\left(R_p\right)\right| \le & \left\|R_1-U_1\right\|\left\|R_2\right\|\left\|R_3 \right\|\left\|R_4\right\| \\ & +\left\|U_1\right\|\left\|R_2-U_2\right\|\left\|R_3\right\|\left\|R_4\right\| \\ & +\left\|U_1\right\|\left\|U_2\right\|\left\|R_3-U_3^*\right\|\left\|R_4\right\| \\ & +\left\|U_1\right\|\left\|U_2\right\|\left\|U_3^*\right\|\left\|R_4-U_4^*\right\|. \end{aligned} \]

By Lemma ?, we have that \(R_j\) is truly a quadratic approximation for \(U_j\), that is, \(\lVert R_j - U_j \rVert \le \frac{\lVert H_j \rVert^3}{6} \le \frac{\eta^{3}}{6}\) when \(j=1,2\). And likewise we have \(\lVert R_j - U_j^* \rVert \le \eta^{3}/6\) when \(j = 3,4\). Since \(\lVert I \rVert = \sqrt{N}\) and \(\lVert H_j \rVert\) is small for each \(j=1,2,3,4\), then the size of \(R_j\) is close to the size of the identity matrix \(I\), which means that \(\lVert R_j \rVert \le C\) where \(C\) depends only on \(N\). Lastly, we know that \(\lVert U_j \rVert = \sqrt{N}\) for each \(j\), so we obtain from the previous display the new bound \[ \left|\phi\left(U_p\right)-\phi\left(R_p\right)\right| \le C \eta^3, \] as desired.

Notes on Chapter 14 of Klenke

Lemma 14.7 (Klenke (2014)) Let \(\emptyset \neq J \subset I\). Then \(X_J^I\) is measurable with respect to \(\mathcal{A}_I-\mathcal{A}_J\).

Proof. For any \(j \in J\), \(X_j\) is measurable with respect to \(\mathcal{A} - \mathcal{A}_j\) by definition of the product \(\sigma\) algebra, Definition 14.4. But \(X_j = X_j^J \circ X_J^I\) as functions, so therefore we can replace \(X_j\) by \(X_j^J \circ X_J^I\) in the previous sentence. To conclude measurability of \(X_J^I\), we need to apply the following result of Klenke (2014):

Corollary 1.82 (Measurability of composed maps) Let \(I\) be a nonempty index set and let \((\Omega, \mathcal{A}),\left(\Omega^{\prime}, \mathcal{A}^{\prime}\right)\) and \(\left(\Omega_i, \mathcal{A}_i\right)\) be measurable spaces for any \(i \in I\). Further, let \(\left(X_i: i \in I\right)\) be a family of measurable maps \(X_i: \Omega^{\prime} \rightarrow \Omega_i\) with \(\mathcal{A}^{\prime}=\sigma\left(X_i: i \in I \right)\). Then the following holds: A map \(Y: \Omega \rightarrow \Omega^{\prime}\) is \(\mathcal{A}-\mathcal{A}^{\prime}\)-measurable if and only if \(X_i \circ Y\) is \(\mathcal{A}-\mathcal{A}_i\)-measurable for all \(i \in I\).

We want to show that \(X_J^I\) is measurable with respect to \(\mathcal{A}_I-\mathcal{A}_J\). Does \(\mathcal{A}'\) in the corollary correspond to \(\mathcal{A}_J\)? For that we would need that \(\mathcal{A}_J = \sigma(X_j^J: j \in J)\). Luckily this is true by inspection of Definition 14.4. Definition 14.9 (Cylinder sets Klenke (2014)) For any \(i \in I\), let \(\mathcal{E}_i \subset \mathcal{A}_i\) be a subclass of the class of measurable sets.
